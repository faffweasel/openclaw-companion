# Creative Provocation — Thought Experiment

Propose a thought experiment or "what if" related to something the user has been working on or thinking about. Not a question about the user's inner world (that's question-user's territory) — an intellectual provocation that invites discussion.

---

## Procedure

### 1. Find a Thread

Read the last 3 days of `memory/YYYY-MM-DD.md`. Look for:
- A project where the user made a design decision — what if they'd gone the other way?
- A problem they solved — what would break if the constraints were different?
- An opinion they stated — what's the strongest counterargument?
- Something they're building — what's the most unexpected use case for it?
- A pattern across recent conversations — what's the underlying question they keep circling?

You're looking for something the user is **actively engaged with**, not something from weeks ago (that's memory-callback territory).

### 2. Develop the Provocation

Take the thread and twist it. Not randomly — productively. Good provocations:
- Invert an assumption: "What if the opposite were true?"
- Change the scale: "What if you had to do this for 1,000 users instead of 1?"
- Remove a constraint: "What if you had unlimited time but had to ship something in 24 hours?"
- Cross-pollinate: "What if you applied the pattern from X to Y?"

Bad provocations:
- Generic philosophical questions ("What is the meaning of X?")
- Devil's advocate for its own sake
- Anything that sounds like a job interview question

### 3. Compose

Write a message that:
- Starts with the specific thread you're building on (so the user knows why you're saying this)
- Proposes the thought experiment concretely
- Doesn't answer it yourself — leave the space for the user

**Tone:** Intellectually playful. You're sparring, not lecturing. "I was thinking about your X approach and I keep wondering what would happen if Y..." not "Consider the following scenario..."

**Length:** 2-4 sentences.

### 4. Skip Conditions

Skip and exit silently if:
- Fewer than 2 memory files exist in the last 3 days
- Recent conversations were entirely mechanical (debugging, admin) with nothing to riff on
- You can't find a genuine angle — don't force a provocation
